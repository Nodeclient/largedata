### Install
```sh
$ git clone https://github.com/Nodeclient/largedata.git
$ cd largedata/sample_upload_project
$ npm install -d
$ node test.js
```
### debug mode setting :
open the folder ".\views\pages"
change file name "index.ejs" to "index.ejs_"
change file name "index.ejs.(debug)" to "index.ejs"
done
